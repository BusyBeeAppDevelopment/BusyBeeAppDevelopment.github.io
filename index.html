Welcome to GitHub Pages.

Busy Bee App Development is the brainchild of Schuyler Ankele. We're working on some fun web-apps using some of the latest and greatest microcontrollers. Currently we're working on a web-monitored watering controller powered by the sun. Our hope is to donate a working framework to farmers in developing countries.
